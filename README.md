# IKappaID PinkSlip

Vehicle title system for Project Zomboid Build 42. Claim vehicles into portable Pink Slip items, trade them, and redeploy later with full state capture.

- **Version:** 0.2.12
- **Author:** IKappaID
- **Workshop:** IKappaID's Pink Slips

## Features

- Total state capture (engine, parts, fuel, trunk contents)
- UID continuity across claim/deploy cycles
- Server-authoritative multiplayer
- Radial menu claim and quick-deploy hotkey
